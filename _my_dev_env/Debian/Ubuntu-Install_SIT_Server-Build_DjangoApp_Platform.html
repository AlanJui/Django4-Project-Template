      # 在 Ubuntu 建置 Django App 作業平台指引

      # 摘要

      在 VirtualBox VM 執行安裝作業，使 VM 成為 Django App 之「作業平台」。

      • 安裝開發作業基礎工具
      • 安裝 Node.js 作業環境
      • 安裝 Python 作業環境
      • 安裝編輯器
      • 安裝 nginx 服務
      • 安裝 Django App 原始碼
      • 安裝 uWSGI 服務

      # 安裝開發作業基礎工具

      1. 安裝 git 版本管理套件。
      sudo apt install git
      1
      1. 安裝檔案下載工具。
      sudo apt install curl

      1. 啟動 uwsgi 服務
