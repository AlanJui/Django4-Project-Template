# 透過 WordPress 驗證 docker-compose 已能正常運作

（1）設定 WordPress 用 docker-compose.yml 組態檔

$ vim docker-compose.yml

docker-compose.yml

wordpress:
  image: wordpress
  links:
    - db:mysql
  ports:
    - 80:80

db:
  image: mariadb
  environment:
    MYSQL_ROOT_PASSWORD: Passw0rd

（2）啟動 docker 容器

$ sudo docker-compose up -d

（3）驗證 docker 容器已啟動，並能正常運作
