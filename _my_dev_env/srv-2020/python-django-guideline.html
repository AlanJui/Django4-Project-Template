      # 建置 Pyton 作業環境

      # 摘要

      建置 Python 語言開發應用系統時，所需之作業環境。

      主要作業：

      • 使用 pyenv 管理 Python 版本
      • 使用 Poetry 管理 Python 套件

      # 安裝 pyenv 作業

      1. 自 GitHub 安裝 pyenv

        自 GitHub 下載 pyenv 及 pyenv-virtualenv

        cd
git clone https://github.com/pyenv/pyenv.git ~/.pyenv
git clone https://github.com/pyenv/pyenv-virtualenv.git ~/.pyenv/plugins/pyenv-virtualenv
        1
        2
        3

      2. 設定 Shell 作業環境

        vim ~/.profile
        1

        ~/.profile:

        ......
#-----------------------------------------------------------
# pyenv
#-----------------------------------------------------------
export PYENV_ROOT="$HOME/.pyenv"
export PATH="$PYENV_ROOT/bin:$PATH"

if command -v pyenv 1>/dev/null 2>&1; then
  eval "$(pyenv init -)"
  eval "$(pyenv virtualenv-init -)"
fi

# export PYENV_VIRTUALENV_DISABLE_PROMPT=1
export PIPENV_VERBOSITY=-1
......

      # poetry 作業指引

      # 安裝 poetry

      1. 安裝軟體。

        curl -sSL https://install.python-poetry.org | python3 -
        1

      2. 設定 Shell 執行路徑。

        (a) 編輯 Shell 設定檔： ~/.bashrc

        nvim ~/.bashrc
        1

        (b) 加入設定

        . . .
export PATH="/home/sammy/.local/bin:$PATH"
        1
        2

        (c) 重啟 Shell

        source ~/.bashrc
        1

      3. 驗證已安裝成功

        poetry --version
        1

      # 管理 python 虛擬環境

      # 建置 python 虛擬環境

      建置專案使用之 Python 虛擬環境。

      cd ~/workspace/django/<ProjectRoot>/
pyenv local <VerNo>
python -m venv .venv
      1
      2
      3

      # 啟用專案虛擬環境

      1. 透過 poetry 啟動專案虛擬環境。

        poetry shell
        1

      2. 安裝 python 套件管理工具： pipenv 。

        poetry add <PackageName>
poetry add <PackageName> --group dev
        1
        2

      # 安裝 Python 套件

      poetry install
      1
